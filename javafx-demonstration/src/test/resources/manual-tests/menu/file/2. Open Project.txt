
1. The application has no project opened.
    1.1 Start the open file dialog.
        1.1.1 The OK button should be disabled if the compulsory fields (title and path) are not filled or the project is invalid.
        1.1.1 The Error message should be displayed if the compulsory fields (title and path) are not filled or the project is invalid. 
2. The application has projected opened but not modified.
        Same as section 1.
3. The application has projected opened and modified.
        3.1 A popup dialog shows with OK and Cancel buttons.
        3.2 If OK is clicked. Save the old project and go to section 1.
        3.3 If Cancel is clicked, close the dialog and do nothing.
